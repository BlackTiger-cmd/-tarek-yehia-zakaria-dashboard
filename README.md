# 👨‍💻 الملف الشخصي للمطور | طارق يحيى زكريا حسن الاسود

مرحباً بك في المنصة الرسمية لتوثيق المشاريع البرمجية الخاصة بالمطور **طارق يحيى زكريا حسن الاسود**. تم تصميم هذا الموقع ليكون المرجع التقني المعتمد لمراجعي التطبيقات والشركات التقنية.

---

### 🚀 معلومات المطور (Developer Identity)
* **الاسم الرباعي:** طارق يحيى زكريا حسن الاسود
* **الفئة:** مطور تطبيقات أندرويد مستقل (Individual Developer)
* **التخصص:** بناء تطبيقات خدمية، إدارة قواعد بيانات Firebase، وأنظمة تتبع العمليات.

### 📱 أبرز المشاريع (Key Projects)
1. **Black Tiger Home Services:** تطبيق شامل لإدارة الخدمات المنزلية واللوجستية.
2. **Order Logic Engine:** نظام منطقي لإدارة دورة حياة الطلبات وضمان الجودة.
3. **Biometric Security Module:** وحدة برمجية مخصصة لتأمين تسجيل الدخول بالبصمة.
